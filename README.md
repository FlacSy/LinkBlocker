# Link Blocker

## Описание
Link Blocker - это инструмент для блокировки нежелательных ссылок в текстовых сообщениях. Этот скрипт разработан на языке Python и использует Telegram API для обнаружения и блокировки определенных ссылок.

## Требования
Прежде чем начать использовать Link Blocker, убедитесь, что у вас установлен Python версии 3.10 и выше.

## Установка
1. Установите необходимые библиотеки, выполнив следующую команду в терминале:
    ```bash
    pip install -r requirements.txt
    ```

2. Создайте файл `.env` и вставьте туда свой токен Telegram. Пример файла `.env`:
    ```plaintext
    TELEGRAM_TOKEN=45349503445024:BBhiufnerufgheurfgHGHfgergGG
    ```

## Настройка
Если вы хотите добавить дополнительные ресурсы для блокировки, откройте файл `links.yaml` и добавьте необходимые ссылки туда.

Пример файла `links.yaml`:
```yaml
- http://example.com
- https://spam-site.net
- ftp://unwanted-resource.org
```

## Запуск
Запустите скрипт, чтобы начать мониторинг и блокировку ссылок в текстовых сообщениях.

```bash
python main.py
```
